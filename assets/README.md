# Assets Ordner

Dieser Ordner enthält alle Bilder und Mediendateien für die Webseite.

## Benötigte Dateien

### Logo
- **logo.png** - Ihr Firmenlogo (empfohlene Größe: 200x60px oder höher für Retina-Displays)
  - Aktuell wird `logo-placeholder.svg` als Platzhalter verwendet

### Optionale Bilder

Für eine noch professionellere Webseite können Sie folgende Bilder ergänzen:

1. **hero-bg.jpg** - Hintergrundbild für die Hero-Section
   - Empfohlene Größe: 1920x1080px
   - Motiv: z.B. Luxusfahrzeug, Werkstatt, oder abstraktes Automobil-Motiv

2. **team-fotos** - Fotos der Mitarbeiter
   - Format: Quadratisch (z.B. 400x400px)
   - Für die "Über uns" Seite

3. **service-images** - Bilder für die einzelnen Leistungen
   - Optional zur Aufwertung der Leistungsseite

## Bildformate

- **Logo**: PNG mit transparentem Hintergrund oder SVG
- **Fotos**: JPG oder WebP für beste Performance
- **Icons**: SVG für scharfe Darstellung in allen Größen

## Optimierung

Vor dem Hochladen sollten alle Bilder optimiert werden:
- Komprimierung für schnellere Ladezeiten
- Responsive Größen für verschiedene Geräte
- WebP-Format für moderne Browser
